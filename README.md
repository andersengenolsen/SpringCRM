# SpringCRM

Application built with Spring MVC.

Simple CRM application. User must authenticate (Spring Security), before adding / deleting customers. Customers are saved to a MySQL database.

Key dependencies:

    Spring MVC
    Spring Security
    Hibernate Core
    Hibernate Validator
